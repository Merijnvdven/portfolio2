Javascript simpele POC

Opdracht

Ik heb de opdracht gekregen om een simpele POC te maken waarin je kan zien dat ik de basis van Javascript begrijp. In deze opdracht heb ik 3 knoppen gemaakt met elk een verschillende functie. 1 button om de achtergrond te veranderen. 1 button om een andere button te laten verdwijnen. En 1 button om de kleur van de button te veranderen.

Bekijk hier

Wat heb ik geleerd?

Met gebruik van javascript functies toewijzen aan knoppen. Ik weet nu hoe ik een actie moet laten gebeuren door middel van op een element te klikken. In dit geval een button bijvoorbeeld een andere kleur geven. Ook heb ik met if en else statements gewerkt. Dat was in dit geval handig voor als je nogmaals op de knop drukt om je verandering weer ongedaan te maken.

Wat kon beter?

Ik heb zelf heel lang proberen uit te vogelen hoe de knoppen werkten. Ik had eerder op internet naar voorbeelden kunnen zoeken om een hoop tijd te besparen.
