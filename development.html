Javascript simpele POC

Bekijk hier

Uitleg

Frank heeft na het bekijken van mijn site als feedback meegegeven om een simpele POC te maken waarin je kan zien dat ik de basis van Javascript begrijp. Dat heb ik hier dus ook gedaan. Als je op de link klikt kom je op een scherm terecht met 3 knoppen. Elke knop heeft een verschillende functie. Als je op "maakrood" klikt wordt de achtergrond van de pagina rood. Klik je op de verdwijn knop verdwijnt de "maakrood knop". Klik je er nogmaals op komt de knop weer tevoorschijn en wordt de achtergrond gereset. Druk je op "buttonblauw" wordt de kleur van die button donkerblauw. De opzet van deze POC is vrij simpel, maar dit heeft mij toch verder ontwikkeld in het developpen, omdat ik nu meer kennis heb opgedaan in het gebruiken van javascript. Wat ik met name hiervan heb geleerd is het toewijzen van functies aan elementen.
